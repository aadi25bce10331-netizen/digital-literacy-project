# Task 2 – Student Digital Portfolio

## Platforms Set Up

I set up accounts on the following three platforms: **GitHub**, **LinkedIn**, and **Kaggle**.

---

## What Each Platform Is Used For

**GitHub** is a web-based platform for version control and code hosting. Developers use it to store their code, track changes using Git, collaborate on open-source projects, and showcase their technical work to potential employers. I created a profile README — a special public repository named after my username — that includes my name, branch (CSE), year, and a sentence about what I hope to learn: *"I am excited to build projects in AI and software development during my time at VIT Bhopal."*

**LinkedIn** is a professional networking platform used by employers, recruiters, internship coordinators, and working professionals worldwide. It is the primary platform for job and internship applications in the technology industry. I filled in my Education section with my current B.Tech degree, institution (VIT Bhopal University), branch (CSE), and expected graduation year (2029).

**Kaggle** is a data science and machine learning community platform. It hosts datasets, competitions, and free learning courses in Python, machine learning, SQL, and data visualisation. I created an account to access its free Python introductory course and explore the dataset library.

---

## Future Plans

Over the next four years, I plan to use **GitHub** to host code from every major project and lab assignment, gradually building a portfolio that is visible to internship recruiters. I will use **LinkedIn** to connect with seniors from my branch, follow companies I am interested in, and apply for internships from my second year onward. I will use **Kaggle** to practise data science and participate in beginner competitions once I have completed my first programming course.
