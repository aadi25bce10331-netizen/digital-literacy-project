## Project Overview

This repository is my **Digital Literacy Portfolio** for the CSE0001 course at VIT Bhopal University. As part of the Student Digital Ambassador programme, I have completed five tasks covering the key pillars of digital literacy — visual communication, professional online presence, coding platforms, email etiquette, and cybercrime awareness.

All work is organised in this single GitHub repository. Each folder corresponds to one task and one course module.

## Module Summaries

### Task 1 – Digital Literacy Awareness Infographic (Module 1\)

Created a one-page infographic using **Canva** covering: what digital literacy means, safe internet practices, and the importance of a professional online presence. Exported and added to `task-1-presentation/`.

### Task 2 – Student Digital Portfolio (Module 2\)

Set up accounts on **GitHub**, **LinkedIn**, and **Kaggle**. Created a GitHub profile README with name, branch, year, and learning goals. Updated LinkedIn Education section. Screenshots in `task-2-portfolio/`.

### Task 3 – Coding & Collaboration Platforms (Module 3\)

Completed the **"Solve Me First"** beginner challenge on **HackerRank**. Built a 5-question **Digital Literacy Awareness Quiz** using Google Forms (link below). Screenshots in `task-3-platforms/`.

### Task 4 – Professional Email & Etiquette Guide (Module 4\)

Drafted two professional emails — one requesting a deadline extension and one expressing internship interest. Created a 10-point Social Media Do's and Don'ts checklist. Files in `task-4-email-etiquette/`.

### Task 5 – Cybercrime Awareness: Phishing (Module 5\)

Wrote a 200–250 word case study on phishing. Created a Stay Safe Online checklist with 8+ actionable tips including UPI-specific safety advice. Files in `task-5-cybercrime/`.
