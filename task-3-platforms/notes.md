# Task 3 – Coding & Collaboration Platforms

## Part A – Coding Practice: HackerRank

**Platform:** HackerRank (hackerrank.com)

I created an account on HackerRank and completed the **"Solve Me First"** beginner challenge under the Problem Solving track. The challenge required reading two integers as input and printing their sum — a simple task, but one that introduced me to the HackerRank submission interface, test cases, and the concept of an online judge.

Upon completion, I received a **Problem Solving** skill badge, which is now visible on my HackerRank profile.

## Part B – Google Workspace Collaboration: Digital Literacy Quiz

**Platform:** Google Forms (forms.google.com)

I created a 5-question **"Digital Literacy Awareness Quiz"** intended for my batchmates. The form is linked to a Google Sheet that automatically captures all responses.

### Quiz Questions:

1. **(Multiple Choice)** What does the term "digital literacy" mean?  
     
   - a) Knowing how to type fast  
   - b) The ability to use digital tools safely, responsibly, and effectively ✓  
   - c) Only using social media  
   - d) Knowing how to code

   

2. **(Multiple Choice)** Which of the following is a sign of a phishing email?  
     
   - a) It comes from a known sender with no attachments  
   - b) It contains urgent language and asks you to click a link ✓  
   - c) It has your name in the subject line  
   - d) It is sent from a .edu email address

   

3. **(Multiple Choice)** What should a strong password include?  
     
   - a) Your name and birth year  
   - b) Only numbers  
   - c) A mix of uppercase, lowercase, numbers, and symbols ✓  
   - d) Your mobile number

   

4. **(Multiple Choice)** Which platform is most suitable for showcasing your code projects to employers?  
     
   - a) Instagram  
   - b) GitHub ✓  
   - c) Snapchat  
   - d) Reddit

   

5. **(Short Answer)** Name one safe internet practice that every college student should follow.

## Reflection

Working with HackerRank showed me how online judges evaluate code automatically, which is a completely different experience from writing code locally. Even a simple problem like "Solve Me First" helped me understand how to read input correctly in a competitive programming environment.

Google Forms was surprisingly powerful for building a professional-looking quiz in minutes. The automatic Google Sheet integration means I could actually deploy this quiz with my batchmates and analyse the results without any extra setup. Both tools will be directly useful in my academic life — HackerRank for practising programming, and Google Forms for conducting surveys, quizzes, and project feedback forms throughout my degree.
